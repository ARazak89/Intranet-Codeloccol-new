### Polices
- **Famille principale** : `'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif`
- **Taille de base** : `16px` (1rem)
- **Hauteur de ligne** : `1.6`

### Hiérarchie des Titres
- **H1** : `2rem` (32px) - `font-weight: 700`
- **H2** : `1.75rem` (28px) - `font-weight: 700`

- **H5** : `1.125rem` (18px) - `font-weight: 600`

### Tailles de Texte
- **Large** : `1.25rem` (20px)
- **Base** : `1rem` (16px)
- **Small** : `0.875rem` (14px)
- **Extra Small** : `0.75rem` (12px)

## 🔲 Espacement

### Système d'Espacement (rem)
- **xs** : `0.25rem` (4px)
- **sm** : `0.5rem` (8px)
- **md** : `1rem` (16px)
- **lg** : `1.5rem` (24px)
- **xl** : `2rem` (32px)
- **2xl** : `3rem` (48px)
- **3xl** : `4rem` (64px)

### Padding et Margin
- **Padding Standard** : `1rem` pour les cartes, `1.5rem` pour les sections
- **Margin Standard** : `1rem` entre les éléments, `2rem` entre les sections
- **Gap Grid** : `1.5rem` pour les grilles principales

## 📱 Responsive Design

### Breakpoints
- **Mobile** : `< 768px`
- **Tablet** : `768px - 1024px`
- **Desktop** : `> 1024px`

### Grilles Responsives
```css
/* Desktop */
.grid-4 {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    gap: 1.5rem;
}

/* Tablet */
@media (max-width: 768px) {
    .grid-4 {
        grid-template-columns: repeat(2, 1fr);
        gap: 1rem;
    }
}

/* Mobile */
@media (max-width: 480px) {
    .grid-4 {
        grid-template-columns: 1fr;
        gap: 1rem;
    }
}
```

## 🎯 États et Interactions

### États des Boutons
- **Normal** : Couleur de base
- **Hover** : Légèrement plus foncé, ombre portée
- **Active** : Plus foncé, ombre intérieure
- **Disabled** : Opacité 50%, curseur non autorisé

### États des Cartes
- **Normal** : Ombre légère
- **Hover** : Translation vers le haut (-2px), ombre plus prononcée
- **Focus** : Bordure colorée (pour l'accessibilité)
