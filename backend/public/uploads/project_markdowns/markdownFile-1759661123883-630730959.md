# Front-end Style Guide - Single Price Grid

## Layout

The layout features a card-based design, centered on the page, and responsive to stack elements vertically on smaller screens.

### Breakpoints

- Mobile: `max-width: 768px`

## Colors

### Primary

- **Call to Action Green**: `#1eb8a8`
- **Why Us Green**: `#43d6c7`
- **Highlight Yellow**: `hsl(71, 73%, 54%)`
- **Button Green**: `#c4f13b`
- **Button Hover Green**: `#b3df2f`

### Neutral

- **Background**: `#eaf3f7`
- **Card Background**: `#fff`
- **Header Text Dark Grey**: `#333`
- **Header Paragraph Grey**: `#666`
- **White**: `#fff` (for plan and why us sections)

## Typography

### Font Family

- `Arial, sans-serif`

### Font Sizes

- **Paragraphs and List Items**: `16px`
- **Header H2**: Default browser size
- **Plan H3**: Default browser size
- **Price**: `2rem`
- **Per Month Text**: `1rem` (inline style)
- **Button**: `1rem`

### Font Weights

- **Body**: `bold` (for `.highlight` class)
- **Price**: `bold`

## Interactivity

- **Button Hover**: Changes background color.
